## API

The package is available by importing its default function:

```js
import GitHub from '@rqt/github'
```

%~%

```## constructor => GitHub
[
  ["token?", "string"]
]
```

Create a new instance of a GitHub API client. If the token is given, it will be used for methods that require it.
